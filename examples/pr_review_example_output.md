## What Gets Posted Where

### GitHub PR Comment
- Full review with code blocks and formatting
- Linked to specific line numbers (when possible)
- Markdown format

### Jira Ticket Comment
- Same content as GitHub
- Jira markup format (h1., {code}, {panel})
- Includes link to PR

### Ticket Status Changes

| Review Result | Status Change |
|--------------|---------------|
| APPROVE | Stays in "In Review" |
| REQUEST CHANGES | Stays in "In Review" |
| BLOCK | Moved to "In Development" |

### Labels Added

- `ai_pr_reviewed` - Always added after review
- `pr_review_wip` - Removed if present
